The **Archaeal Proteome Project (ArcPP)** is a *community-driven scientific initiative* focused on characterizing and understanding the **proteomes of archaea** the full set of proteins expressed by archaeal organisms using large-scale proteomics and bioinformatics.

**🔬 Core concept**

* ArcPP collects and *reanalyses mass spectrometry (MS) proteomics datasets* for archaeal species, especially the model archaeon *Haloferax volcanii*.
* Instead of individual studies interpreting MS data separately, ArcPP integrates diverse datasets using unified, state-of-the-art computational workflows. This produces a comprehensive, high-confidence catalogue of archaeal proteins.

**📊 What it does**

* Reprocesses raw MS data from multiple experiments and conditions.
* Applies advanced peptide spectrum matching and stringent quality control to maximize proteome coverage.
* Produces an interactive database of reliably identified archaeal proteins that researchers can *query, compare and build on*.

**🧬 Process**
• Collect public mass spectrometry (MS) proteomics datasets

• Compile and prepare a unified proteome reference database

• Optimise search parameters for peptide identification

• Perform peptide spectrum matching across datasets

• Control false discovery rates (FDRs)

• Combine all reanalysed datasets

• Annotate biological features

• Extract biological insights

• Make results available to the community
